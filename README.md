# bookHub

# Live Link: https://bookhub-4ie9.onrender.com/

## Description
BookHub is an online book borrowing platform built using the Django MVT pattern. The frontend is styled with Bootstrap.

## Key features:
- User Registration and Login: Users need to register and log in to access the site.
- Book Collection: Browse and view details of available books.
- Account Deposits: Users can deposit money into their account.
- Book Borrowing: Users can borrow books using their deposited funds. Borrowing is restricted if the account balance is insufficient. An email confirmation is sent upon borrowing a book.
- Refunds: Refunds are issued to usersâ€™ accounts after returning books.
- Book Reviews: Users can review books upon return.
- Borrowed Books: Users can view a list of borrowed books in their profile.
- Transaction History: Users can access their transaction history on their profile page.
